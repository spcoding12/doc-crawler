# Supported languages

SonarQube Server provides analysis of different languages depending on the edition youâ€™re running.

SonarQube Server is an automated code review and static code analysis solution for several languages to satisfy your unique needs. Support for your language may vary depending on the SonarQube Server edition you are running.

The table below lists the supported languages. For the supported versions and language-specific properties, refer to each language page directly. See _Related pages_ below.

Infrastructure as Code (IaC) analysis is supported for the following languages listed in the table below: Ansible, Azure Resource Manager, CloudFormation, Docker, Kubernetes, Terraform.
